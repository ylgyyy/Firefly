还在为远程管理无网设备、调试 BIOS 或重装系统而烦恼？传统的远程控制软件往往受限于操作系统和网络环境。今天，我们就部署一款硬件级的远程管理神器——**OneKVM**。它通过**视频采集卡+USB对拷**的纯硬件方式，实现了**无视被控端系统、无需安装软件、甚至完全脱离网络**的极致远程控制。无论是服务器维护、设备调试还是系统安装，OneKVM 都能让你像坐在设备前一样直接操作。接下来，就让我们将这款强大的工具，部署到性价比极高的“矿渣”玩客云上。

## **准备工作** **🛠️**

1. **矿渣玩客云**一台（用于部署OneKVM主机）
2. **OneKVM固件**（开源免费的远程管理工具）
3. **USB_Burning_Tool**（晶晨芯片烧录工具）
4. **VC运行库合集**（确保烧录工具正常运行）
5. **USB对拷线**（用于刷机和连接被控端）
6. **USB视频采集卡**（采集被控端的HDMI视频画面）
7. **一把镊子**（用于短接刷机触点）
8. **一个拆机棒**（用于撬开玩客云外壳）
9. **一把十字螺丝刀**（用于拆卸螺丝）

## **第一步：下载 OneKVM 固件** **📥**

访问 OneKVM 的 GitHub 发布页：
https://github.com/mofeng-git/One-KVM/releases

在页面中找到 **“image 251001”** 版本，下载 One-KVM_by-SilentWind_Onecloud_250930.burn.img.xz 文件。下载完成后，使用解压工具（如7-Zip）将其解压，得到 .img 格式的刷机固件。

## **第二步：下载并安装 USB_Burning_Tool** **⚙️**

1. 访问官网下载烧录工具：
   https://androidmtk.com/download-amlogic-usb-burning-tool
2. 下载 **v2.2.0** 版本，解压并安装。
3. 首次运行时，若出现 "**找不到 mfc100.dll**" 等报错，请下载并安装 **VC运行库合集**：
   下载链接：https://wwbsy.lanzoue.com/iXABS3huoysh
   密码：8cdv
4. 安装运行库时，一直点击“下一步”即可。安装完成后，再次打开 USB_Burning_Tool，此时应能正常启动。

## **第三步：拆解玩客云** **🔧**

**提示**：若你的玩客云**已刷过机**，则无需拆机短接。只需按住 **Reset** 复位键，再插入电源即可连接刷机软件。若为**未刷机**的原版系统，请按以下步骤操作。

1. 使用拆机棒撬开玩客云的**第一层后盖**。
2. 用十字螺丝刀卸下露出的 **6颗螺丝**。
3. 再用拆机棒撬开**第二层后盖**，即可抽出主板。
4. 玩客云主板主要有两种版本：

- **新版**：SD卡槽旁印有 **“V1.3”** 字样。
- **旧版**：SD卡槽旁无任何标识。
- 两个版本的**刷机短接触点位置不同**，请参考下图确认你的主板版本并进行短接。

## **第四步：刷入 OneKVM 固件** **🔄**

1. 将对拷线一端插入玩客云**靠近HDMI接口的USB口**，另一端插入电脑。
2. 用镊子**短接主板上的刷机触点**（勿松手）。
3. 此时 USB_Burning_Tool 应识别到设备（如“连接成功”提示）。**识别成功后即可移开镊子**。
4. 点击软件上的 **“文件” → “导入烧录包”**，选择第一步解压得到的 .img 固件文件。
5. 固件校验完成后，点击 **“开始”** 按钮。
6. 当听到电脑发出 **USB设备连接的“叮咚”提示音**时，**立即拔插一次**玩客云上的对拷线。

若未及时拔插，工具可能报错，此时再拔插仍可继续刷机。

1. 等待刷机进度完成，提示“烧录成功”。关闭软件，将玩客云按原样组装回去。

## **第五步：试机启动** **✅**

将玩客云的HDMI连接到显示器 并接通电源。当代码跑完屏幕上显示出**Onecloud画面**，则表示固件刷入成功，设备启动正常。

## **第六步：连接被控端设备** **🔗**

1. 拔掉连接显示器的HDMI线。
2. 为玩客云插入**网线**。
3. 在玩客云**网口一侧的USB口**上插入 **USB视频采集卡**，接入HDMI视频线。
4. 在玩客云**靠近HDMI接口的USB口**上插入 **USB对拷线**
5. 将HDMI视频线的另一端接入被控设备HDMI输出口，USB对拷线的另一端接入被控设备的任意USB口。

至此，硬件连接完成。

## **第七步：OneKVM 初体验** **🖥️**

1. 进入路由器后台的 **DHCP客户端列表**，查找主机名为 **“onecloud”** 的设备，并记录其IP地址。
2. 在浏览器中输入该IP地址，打开 OneKVM Web管理界面。
3. 使用默认账号密码登录（均为 admin）。
4. 登录后，点击 **“KVM”** 选项，即可看到被控设备的实时画面。
5. 尝试重启被控设备，并按下进入BIOS的快捷键，这时就可以**远程进入并操作被控端的BIOS了**，体验硬件级控制的魅力。
6. OneKVM 还支持电源管理、网络唤醒、快捷键发送、挂载ISO镜像等，甚至可以用PE给被控端装个系统。

更多功能可查阅官方文档：
https://docs.one-kvm.cn/python/

## **第八步：修改 OneKVM 账号密码** **🔐**

为安全起见，建议修改默认凭证。

1. 退出当前登录，回到登录页面。
2. 使用 admin 账户登录，点击 **“Terminal”** 进入命令行界面。
3. 输入以下命令切换至root账户：

```
su root
```

提示输入密码时，输入默认密码：1234（输入时无显示）。

1. 为root账户设置新密码：

```
passwd
```

按提示输入并确认新密码。

1. 查看当前用户列表：

```
kvmd-htpasswd list
```

1. 添加一个新用户（例如 ylgyyy）：

kvmd-htpasswd add 你的用户名

设置该用户的密码。

1. 再次查看用户列表，确认新用户已添加。
2. 删除默认的 admin 用户：

```
kvmd-htpasswd del admin
```

1. 退出并使用新创建的用户登录 OneKVM，验证是否成功。

## **第九步：配置内网穿透（实现公网访问）** **🌐**

OneKVM 作者提供了免费的穿透服务。

1. 登录 OneKVM Web界面，点击 **“gostc”**。
2. 新增一个客户端，自定义名称。
3. 在浏览器中打开穿透服务平台：[https://gostc.mofeng.run](https://gostc.mofeng.run/)
4. 注册并登录账号。
5. 点击左侧 **“客户端”**，创建一个与OneKVM同名的客户端，保存后复制生成的**密钥**。
6. 回到 OneKVM 的 gostc 设置页面，粘贴密钥。
7. 在“服务器”栏填写：gostc.mofeng.run
8. 开启 **“自启动”**，保存并**启动客户端服务**。
9. 返回穿透平台页面，刷新客户端状态，这时客户端就已经 **“在线”了**。
10. 在平台完成 **邮箱绑定** 与 **实名认证**。
11. 进入 **“****隧道管理” → “端口转发”**，新增一个隧道。

- 套餐选择：**Oen KVM Web专用**
- 内网配置：填写内网 OneKVM 的 **IP****地址** 与端口 443
- 远程端口：建议留空（自动分配）

保存后，复制生成的 **公网访问地址**。 切换至外部网络，在浏览器中输入 https:// + 公网地址，即可远程访问你的 OneKVM 了！

## **结语**

至此，你已经成功将玩客云打造成了一台功能强大的硬件级远程管理设备。无论被控设备是何种状态，OneKVM 都能让你从容应对。
